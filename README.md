## 🫛 Main Project: Flowbit

### [Flowbit](https://github.com/bean8080/flowbit)

> 작업의 현재 상태뿐 아니라, 상태 변화 이벤트를 기록하여  
> 일이 어떤 흐름으로 진행되었는지 추적하는 작업 관리 시스템

Flowbit은 단순한 Task 관리 도구가 아니라,  
작업의 현재 상태와 상태 변화 이력을 분리해  
작업이 어떤 과정을 거쳐 현재 상태가 되었는지 추적할 수 있도록 만든 프로젝트입니다.

- Java 21 / Spring Boot 기반 백엔드
- React / TypeScript 기반 프론트엔드
- PostgreSQL / Redis / Docker Compose 기반 개발 환경
- AWS EC2 + Nginx + Docker Compose 기반 실도메인 배포
- Task와 TaskEvent를 분리한 이벤트 기반 이력 추적 구조
- Project Timeline / Analysis 기능 구현
- 트랜잭션을 통한 상태 변경과 이벤트 기록의 정합성 보장
